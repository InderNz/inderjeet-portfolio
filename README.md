# Inderjeet Singh — Portfolio Site
Live at: https://inderjeetsingh.nz
Built with React + Vite. Deployed on Netlify.

## Development
npm install
npm run dev        # localhost:5173

## Testing
npx playwright test                    # full suite
npx playwright test --reporter=list    # with details

## AutoSDLC Pipeline
Jira-driven agentic development pipeline.
Tickets flow: To Do → Selected for Development → In Progress → In Test → Completed
Orchestrator: node orchestrator.cjs

## Tech Stack
- React 18 + Vite
- Playwright (E2E testing)
- EmailJS (contact form)
- Netlify (deployment)
